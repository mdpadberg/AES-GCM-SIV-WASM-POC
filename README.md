# AES-GCM-SIV-WASM-POC

## Online demo
https://mdpadberg.github.io/AES-GCM-SIV-WASM-POC/

## Run locally  

### Prerequisites
- rust
- rustup
- cargo

### How to run this project locally
1. run `./install.sh`
2. run `./build.sh`
3. run `./run.sh`
